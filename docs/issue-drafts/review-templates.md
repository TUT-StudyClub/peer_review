# レビュー用の定型文セット（ドラフト）

レビュー入力でボタンから挿入できる「実用レベル」の定型文セットを定義する。
対象: `frontend/src/app/assignments/[assignmentId]/page.tsx` のレビューコメント入力。

## 定型文セット

- 良い点: `【良い点】主張が明確で、根拠となるデータ/引用が添えられています。`
- 改善点: `【改善点】この部分の論拠が弱いので、出典や具体例を1〜2個追加してください。`
- 根拠不足: `【根拠不足】結論と根拠の間に飛躍があります。データ/引用/図表などのエビデンスを補ってください。`
- 誤字脱字: `【誤字】タイポや表記ゆれがあります。最終提出前に再確認してください。`
- 構成: `【構成】段落ごとに主張→根拠→結論の流れを意識すると読みやすくなります。`
- 具体例: `【具体例】読者がイメージできる具体例を1つ追加すると説得力が増します。`

## 備考
- 挿入時はレビューコメント末尾に改行付きで追記する。
- ボタン押下回数をコンソールログに残す（最低限の計測）。***
